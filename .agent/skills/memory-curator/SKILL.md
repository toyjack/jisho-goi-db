# Memory Curator

## Goal
- プロジェクトの再利用可能な知見を `.agent/memory/` に整理して蓄積する

## Instructions
1. `.agent/memory/MEMORY.md` を読み、今回のタスクに関係する項目を把握する
2. 変更や学びがあれば追記する
   - 決定事項は decisions.md
   - 再発防止と地雷は pitfalls.md
   - 用語と命名は glossary.md
3. MEMORY.md には索引としてリンクだけを追加する
4. 推測は書かない。根拠としてファイルパスかコマンドを添える

## Output format
- 箇条書き
- 1行1要点
- なぜ必要かを1行で添える
