# Memory Protocol

- 作業開始時に必ず `.agent/memory/MEMORY.md` を読み、今回のタスクに関係する前提と地雷を短く列挙してから計画を作ってください。

- MEMORY.md は索引として保ち、詳細は decisions.md / pitfalls.md / glossary.md に書いてください。
- 推測は決定事項として保存しないでください。根拠としてファイルパスかコマンドを添えてください。
